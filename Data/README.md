# Medical Dataset

This folder contains the primary medical dataset used for the development
of the AI-based symptom checker system.

## Dataset Description
The dataset is composed of structured and semi-structured medical information
derived from reliable public medical sources.  
It includes:
- Disease names and descriptions
- Associated symptoms
- Diagnostic and clinical information
- Medical terminology used for symptom–disease mapping

## Usage
The dataset is used as the core knowledge source for:
- Symptom interpretation
- Disease matching
- Retrieval-Augmented Generation (RAG) pipelines
- Medical response generation and validation

## Data Processing
The data has been:
- Collected via web scraping and manual curation
- Cleaned and normalized
- Structured to ensure consistency and usability within AI pipelines

⚠️ Disclaimer:  
This dataset is intended for research and educational purposes only and
must not be used for clinical diagnosis or medical decision-making.
