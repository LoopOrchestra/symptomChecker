# Medical Documentation

This folder contains medical reference documents used to support
the development of the AI-based symptom checker.

## Contents
The documents include:
- Disease overviews (e.g. cancer types, leukemia, eye diseases)
- Diagnostic and clinical descriptions
- Medical terminology and classifications
- Reference material used for data understanding and validation

## Purpose
These files are used for:
- Medical background research
- Validation of symptoms–disease relationships
- Supporting the design of the knowledge base for the RAG system

⚠️ Disclaimer:  
This material is used for research and educational purposes only and
does not constitute medical advice.
